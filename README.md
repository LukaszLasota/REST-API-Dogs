## How to run the app

1\. Clone the repository:

```
git clone https://github.com/LukaszLasota/REST-API-Dogs
```

2\. Install the dependencies:

```
npm install
```

3\. Run dev server

```
npm start
```

## Technologies
Project is created with:
* Webpack 
* HTML 5
* CSS 3/SCSS
* JavaScript ES 6
* Rest API
